# Project Aplikasi Kasir Coffee Shop
Aplikasi kasir desktop menggunakan **Java JDK 21** dan database **MySQL**.

## Cara Install:
1. Import `db_coffeeshop.sql` ke phpMyAdmin.
2. Buka project ini di NetBeans IDE.
3. Pastikan sudah menambahkan Library `MySQL JDBC Driver` dan `JFreeChart`.
4. Jalankan file `login.java`.
